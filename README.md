## Installasi

#### Install toko-buku dengan [composer](https://getcomposer.org/) & [npm](https://npmjs.com/)

- composer

```bash
composer update
```

- npm

```bash
npm install
npm run dev
```

#### Edit file .env

Edit file .env sesuai settingan sendiri

#### Migrasi Database

- php artisan

```php artisan
php artisan migrate
```
